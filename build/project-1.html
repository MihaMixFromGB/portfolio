Интернет-версия

Обзор проекта

Chat for Everyone это приложение, которое позволяет каждому создавать свой чат или несколько чатов, приглашать друзей, коллег или партнеров по бизнесу для общения посредством отправки текстовых сообщений.

Авторизация выполнена с помощью серверной службы Firebase Authentication посредством ввода адреса электронной почты и пароля. Для хранения данных пользователя, чатов и сообщений используется NoSQL облачная база данных Firebase.
