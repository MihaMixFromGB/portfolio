Projetc Overview

Chat for Everyone is the app that Everyone can create a chat or many chats, invite friends, coworkers or business partners for text messaging.

User authentication was made using the Firebase Authentication SDK by submitting an email and password. Firebase Realtime Database (NoSQL cloud database) is used to store user profiles, chats and messages.

There was created a test user for review: student@mail.ru / 123456.
