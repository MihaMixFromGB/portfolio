Project Overview

The online clothing store is created using the website template from Figma. An adaptive layout for four pages is made. There are main, catalog of man's clothing, cart, registration page. It's made for desktop, tablet and mobile.

Menu and other drop-down lists are made using HTML and CSS without Javascript.

The website template - https://www.figma.com/file/7XhT5EWfZX0W6ohgd57Pny/shop.
